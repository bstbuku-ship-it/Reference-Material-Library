# 📚 素材参考库 v1.0

> 一款专为内容创作者设计的本地灵感素材管理工具，单文件 HTML，无需安装，直接浏览器打开即用。

\---

## ✨ 功能特色

* 🗂️ **素材分类管理** — 自定义分类目录，卡片式瀑布流展示
* 🔗 **多平台链接归档** — 支持 Facebook、YouTube 等平台链接收录
* 🖼️ **图片物理存储** — 图片直接归档到本地文件夹，不依赖云端
* 🗑️ **防丢回收站** — 删除有后悔药，支持永久清除
* 📸 **时光机快照** — 自动定期备份元数据，可回溯历史状态
* 📊 **数据分析洞察** — 平台分布、分类热力、收录趋势一览
* 🌗 **深色 / 亮色主题** — 跟随系统或手动切换
* 🔒 **PIN 码访问锁** — 可选密码保护，防止他人查看
* 🔄 **自动更新检测** — 启动时静默检测新版本，一键下载

\---

## 🚀 使用方法

1. 下载最新版本的 `素材参考库\_vX.X.html` 文件
2. 用 **Chrome 或 Edge** 浏览器打开（推荐，支持本地文件夹授权）
3. 点击「选择/连接文件夹」授权一个本地文件夹用于存储图片
4. 开始添加素材 🎉

> ⚠️ 建议使用 \*\*Chrome / Edge\*\* 以获得完整的本地硬盘存储功能。Safari 和 Firefox 会降级到浏览器沙盒缓存模式。

## 🔄 版本更新

软件启动后会自动检测 GitHub 是否有新版本。有更新时会弹窗提示，点击「立即下载」即可获取新文件。旧文件的本地数据文件夹无需迁移，重新连接即可继续使用。

## 📁 数据说明

所有数据保存在你本地授权的文件夹中：

```
你选择的文件夹/
├── database.json       # 素材元数据（标题、标签、链接等）
├── workspace.json      # 工作台数据
└── \[分类名称]/
    └── \[素材名称]/
        ├── 图片1.jpg
        └── 图片2.jpg
```

数据完全本地，不上传任何服务器。
