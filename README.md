# 📚 素材参考库 v1.2

> 一款专为内容创作者设计的本地灵感素材管理工具。**现已升级为浏览器扩展**，安装后常驻工具栏，一键打开，无需每次手动加载文件。

---

## ✨ 功能特色

* 🧩 **浏览器扩展形态** — 点击工具栏图标即可打开，无需再手动拖拽 HTML 文件
* 🗂️ **素材分类管理** — 自定义分类目录，卡片式瀑布流展示
* 🖼️ **图集全览** — 双击素材卡片或点击侧边导航栏，均可快速进入图集全览大图浏览
* 🔲 **九宫格排版预览** — 一键查看多图排版效果
* 🔗 **多平台链接归档** — 支持 Facebook、YouTube 等平台链接收录与解析
* 🖼️ **图片物理存储** — 图片直接归档到本地文件夹，不依赖云端
* 📁 **目录记忆授权** — 自动记住上次连接的本地文件夹，下次打开只需一次授权确认，无需重新选择目录
* 🗑️ **防丢回收站** — 删除有后悔药，支持永久清除
* 📸 **时光机快照** — 自动定期备份元数据，可回溯历史状态
* 📊 **数据分析洞察** — 平台分布、分类热力、收录趋势一览
* 🌗 **深色 / 亮色主题** — 跟随系统或手动切换
* 🔒 **PIN 码访问锁** — 可选密码保护，防止他人查看
* 🔄 **自动更新检测** — 启动时静默检测新版本，一键下载

## 🚀 安装与使用方法

### 安装扩展

1. 下载最新版本的扩展压缩包并解压
2. 打开浏览器（推荐 **Chrome 或 Edge**），进入扩展管理页面：
   - Chrome：地址栏输入 `chrome://extensions`
   - Edge：地址栏输入 `edge://extensions`
3. 打开右上角「开发者模式」
4. 点击「加载已解压的扩展程序」，选择解压后的文件夹
5. 安装完成后，工具栏会出现「素材参考库」图标

### 开始使用

1. 点击工具栏图标打开插件
2. 首次使用点击「选择/连接文件夹」，授权一个本地文件夹用于存储图片和数据
3. 之后每次打开会自动记住该文件夹，只需确认授权即可继续使用
4. 开始添加素材 🎉

> ⚠️ 建议使用 **Chrome / Edge**，以获得完整的本地硬盘直读直写功能。

## 🔄 版本更新

扩展启动后会自动检测是否有新版本发布，有更新时会弹窗提示，按提示下载并重新加载扩展即可。旧版本本地数据文件夹无需迁移，重新连接授权后即可继续使用。

## 📁 数据说明

所有数据保存在你本地授权的文件夹中：

```
你选择的文件夹/
├── database.json       # 素材元数据（标题、标签、链接等）
├── workspace.json      # 工作台数据
└── [分类名称]/
    └── [素材名称]/
        ├── 图片1.jpg
        └── 图片2.jpg
```

数据完全本地，不上传任何服务器。
